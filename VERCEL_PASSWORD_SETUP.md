# 🔐 Vercel 密码保护设置指南

## 📋 功能说明

已为系统添加登录页面保护，只有输入正确密码才能访问系统。

## 🔑 默认密码

**默认密码：** `factory2024`

## ⚙️ 在 Vercel 上设置自定义密码

### 步骤 1：登录 Vercel
访问：https://vercel.com
使用 GitHub 账号登录

### 步骤 2：进入项目设置
1. 在 Dashboard 中找到您的项目
2. 点击进入项目页面
3. 点击顶部的 **Settings** 标签

### 步骤 3：添加环境变量
1. 在左侧菜单选择 **Environment Variables**
2. 添加新的环境变量：
   - **Name**: `VITE_ACCESS_PASSWORD`
   - **Value**: 输入您想要的密码（例如：`MySecretPass123`）
   - **Environment**: 勾选 `Production`、`Preview`、`Development`（全选）
3. 点击 **Save**

### 步骤 4：重新部署
1. 回到项目页面顶部
2. 点击 **Deployments** 标签
3. 找到最新的部署记录
4. 点击右侧的 **⋯** (三个点)
5. 选择 **Redeploy**
6. 点击确认

### 步骤 5：测试
等待 1-2 分钟部署完成后：
1. 访问您的网站
2. 应该会看到登录页面
3. 输入您设置的密码
4. 成功登录！

---

## 🔄 本地开发环境设置

如果您在本地开发时也想测试密码保护：

### 方法 1：使用默认密码
不做任何配置，直接使用默认密码 `factory2024`

### 方法 2：自定义密码
1. 在 `frontend` 文件夹中创建 `.env` 文件（如果没有的话）
2. 添加以下内容：
   ```
   VITE_ACCESS_PASSWORD=您的密码
   ```
3. 重启开发服务器（`npm run dev`）

## 🛡️ 安全提示

1. **设置强密码**：建议使用包含大小写字母、数字和特殊字符的密码
2. **定期更换**：定期在 Vercel 上更新密码
3. **不要泄露**：不要将密码上传到 GitHub 或公开场合
4. **妥善保管**：将密码分享给需要访问系统的同事

## ❓ 常见问题

### Q: 忘记密码怎么办？
A: 登录 Vercel，在项目设置的 Environment Variables 中查看或修改密码

### Q: 可以设置多个用户不同密码吗？
A: 当前版本只支持一个共享密码。如需多用户管理，需要升级到方案2（完整登录系统）

### Q: 登录后会话多久过期？
A: 关闭浏览器标签页后需要重新登录，同一浏览器其他标签页仍可访问

### Q: 修改密码后需要重新部署吗？
A: 是的，修改环境变量后必须重新部署才能生效
